# National Flag Creator

This project is a simple HTML and CSS implementation to create a stylized representation of a national flag, accompanied by a rotating wheel in the center.

## Demo

You can see a live demo of the project https://curious-medovik-53dc3c.netlify.app/



## Features

- Stylized representation of a national flag using HTML and CSS.
- Centered rotating wheel with lines for a decorative effect.
- Easily customizable and extensible for different designs.

## Getting Started

1. Clone the repository:https://github.com/Samanvith20/Indian-flag

    ```bash
    git clone 
    ```

2. Open the `index.html` file in a web browser to view the flag and rotating wheel.

## Usage

Customize the project according to your preferences:

- Modify the colors and styles in the `style.css` file to match the national flag you want to represent.
- Adjust the number of lines and their rotations in the `.wheel .line` CSS rules for the rotating wheel.
